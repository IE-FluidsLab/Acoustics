# Acoustics
Acoustic propagation modeling - code and benchmarks (OpenFOAM and Specfem3d)

The content of this directory will be made available once the article “Modeling Underwater Noise Propagation: A Comparative Study of Fully 3D Time-Domain Numerical Strategies” is published. The manuscript is currently under review.

This directory contains two main subfolders:

OpenFOAM
This folder includes the acoFOAM code and the benchmark setup for the Pekeris waveguide test case.

Specfem3D
This folder contains the configuration files and results for the Gaussian Canyon benchmark test case.
